# Starbucks Adaptive Layout

This project was created as a training project focused on responsive layout development.

## Technologies
- HTML5
- CSS3
- Flexbox
- CSS Grid
- clamp()
- pseudoclasses and pseudoelements

## Features
- responsive layout
- animated borders
- background wave effects
